html

Defines the root of an HTML document.

head

Contains metadata, links, scripts, and title of the document.

title

Specifies the title of the document, shown in the browser tab.

body

Contains the content of the HTML document, like text, images, etc.

h1

Defines the largest heading in HTML. There are six levels from h1 to h6.

p

Defines a paragraph of text.

a

Defines a hyperlink to navigate to another page or section.

img

Embeds an image in the page. It is a self-closing tag.

ul

Defines an unordered list, usually with bullet points.

ol

Defines an ordered list, usually with numbers.

li

Defines a list item in either an ordered or unordered list.

div

Defines a container for other elements, used for layout purposes.

span

Defines an inline container used to group text or elements for styling.

form

Defines an HTML form for user input.

input

Defines an input field where the user can enter data.
